STEPS TO FOLLOW :
=================================

1) Open the solution file "AlintaWeb.sln" in visualstudio to load the project.
2) Run the project "WebApi" using IIS express, it will open the url "https://localhost:44346/swagger/index.html".
3) Swagger can be used to perform the following action :
	- POST : create a new customer
	- PUT : Update an existing customer
	- DELETE : remove an existing customer.
	- Get : search customer by name.
